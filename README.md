# 13.3 Day 6: Bài tập về nhà

*Cập nhật tháng 4 năm 2025*

## Yêu cầu chung

- Tạo thư mục `f8-zoom-day-6` và một GitHub repo cùng tên.
- Sau khi hoàn thành mỗi bài, commit với message rõ ràng.
- Tạo `index.html` với danh sách các bài tập kèm liên kết tương ứng.
- Sử dụng lại `reset.css` và `style.css` từ buổi trước.
- Ưu tiên sử dụng đơn vị `rem` và các thuộc tính CSS đã học trước đó.

## 1. Dàn lại bố cục CV bằng Flexbox

### Yêu cầu

- Dựa trên `cv.html` buổi 4, tạo file mới `cv-flex.html`.
- Dùng Flexbox để chia bố cục:
  - Thêm sidebar trái chứa ảnh + thông tin cá nhân.
  - Phần còn lại chứa kỹ năng, kinh nghiệm, học vấn,...
- Áp dụng các thuộc tính Flexbox đã học.

## 2. Dàn lại Pricing Table bằng Flexbox

### Yêu cầu

- Dựa trên `pricing-table.html` buổi 5, tạo file mới `pricing-flex.html`.
- Dùng Flexbox đặt 3 khối giá ngang hàng.
- Canh giữa nội dung trong từng khối bằng Flexbox.
- Đảm bảo các khối có kích thước bằng nhau, cách đều.

## 3. Tạo Layout danh sách sản phẩm mới

### Yêu cầu

- Tạo file `product-list.html`.
- Dùng Flexbox để tạo layout hiển thị danh sách sản phẩm (6–9 sản phẩm).
- Mỗi hàng có 3 sản phẩm, tự động xuống hàng nếu hẹp.
- Mỗi sản phẩm gồm:
  - Ảnh
  - Tên sản phẩm
  - Giá
  - Nút mua
- Thêm hiệu ứng hover nổi bật cho từng sản phẩm.

## 4. Layout Login – Register

### Yêu cầu

- Tạo `register-layout.html`.
- Bố cục chia làm 2 nửa bằng nhau.
- Bên trái là ảnh minh họa hoặc giới thiệu.
- Bên phải là form đăng nhập/đăng ký.
- Dùng Flexbox để căn giữa form theo chiều ngang và dọc.

## 5. Chinh phục Froggy Pro

### Yêu cầu

- Truy cập: https://froggy-pro.fullstack.edu.vn
- Vượt qua ít nhất 2 pond.
- Chụp ảnh toàn màn hình khi vượt qua.
- Lưu ảnh vào thư mục `/img`.
- Tạo file `froggy.html`.
- Chèn 2 ảnh và mô tả ngắn gọn bên dưới mỗi ảnh.
